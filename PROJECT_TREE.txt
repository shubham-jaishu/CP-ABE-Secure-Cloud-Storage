🎨 KEY FEATURES
═══════════════════════════════════════════════════════════════

✅ Policy-Based Encryption
   • Encrypt files with access policies
   • AND/OR operator support
   • Example: "Role=Doctor AND Department=CSE"

✅ Attribute-Based Decryption
   • Users provide attributes as JSON
   • System evaluates policy match
   • Access granted/denied based on policy

✅ Dual Storage Support
   • Local: Files in ./encrypted_files/
   • S3: Files in AWS S3 bucket
   • Toggle via STORAGE_MODE config

✅ Resource Monitoring
   • CPU usage tracking (psutil)
   • Memory usage tracking
   • Disk usage tracking
   • Operation time logging

✅ Production Ready
   • RESTful API with Flask
   • CORS enabled
   • Comprehensive logging
   • Error handling
   • CloudWatch compatible
   • Docker containerization
